Border




Class .border enables the border across all 4 boundaries
It's like the motherclass for enabling borders

Class .border-top enables the top border

Class .border-bottom enables the bottom border

Class .border-start enables the start border

Class .border-end enables the end border

Substractive Border

Class .border-end-0 removed the end border. Other classes that are used to remove selective borders are border-top-0, border-start-0, border-bottom-0

Rounded Borders

Class for rounded borders are as follows :-
  • rounded (which is used here)
  • rounded-top
  • rounded-end
  • rounded-bottom
  • rounded-start
