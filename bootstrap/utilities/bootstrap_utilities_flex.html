Summary of flex:grow and flex:basis
  • If any div container has flex:grow, it will expand to take up the 100% of remaining column space.
  • flex-basis specifies the width of individual div containers. If flex-basis is 0, minimum column width will be used.
  • Hence flex:grow and flex-basis:0 are a bit contradictory
  • If we are using flex-basis apart from 0 or auto, it is same as specifying the column width/height
